## PWA

Progressive Web Apps(PWA): 모바일 앱과 웹 기술의 장점을 결합한 것

### PWA의 특징

- App스러움
- Push 메시지 기능
- Offline 접속 기능
- 보안

### 요약

사용이 편리하다는 앱의 장점 + 설치 X + 기존 웹의 장점  
웹이지만 앱과 같이 푸시 알림 등의 기능이 가능하고 설치 없이 더 간편하게 접근 가능
